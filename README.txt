# snmp_monitor
一、程序说明:
	基于snmp协议，对交换机端口接受trap告警，根据界面对端口的配置，实时对端口和连线，以及告警码文本进行告警显示或告警恢复
	运行：linux平台，执行bin文件夹下的snmp_pro
	
二、工程说明：
	bin文件夹：可执行程序snmp_pro、图片素材、配置文件config.ini
	src文件夹；源代码及QT工程
	pic文件夹：界面部分截图
	
三、config.ini配置说明：
	[Oid]模块：对每个端口配置相应的oid值，用于程序上电发起get报文获取端口状态，以及运行过程中对trap报文通过oid关联端口号，以在系统端口框架页面对具体端口进行告警
	IP 字段：指定交换机ip
	[WarnText]模块：自定义要关注的告警码、告警名称、告警描述、以及是否对告警描述获取告警的端口并显示，此模块用于配置告警文本区的显示内容
